10.2. Expressions dealing with types

This section describes JSONiq types as well as the sequence type syntax.

10.2.1. Instance-of expressions

Figure 10.1. InstanceofExpr

An instance expression can be used to tell whether a JSONiq value matches a given sequence type.
Example 10.25. Instance of expression
1 instance of integer

Result:
 true

Example 10.26. Instance of expression
1 instance of string

Result:
 false

Example 10.27. Instance of expression
"foo" instance of string

Result:
 true

Example 10.28. Instance of expression
{ "foo" : "bar" } instance of object

Result:
 true

Example 10.29. Instance of expression
({ "foo" : "bar" }, { "bar" : "foo" }) instance of json-item+

Result:
 true

Example 10.30. Instance of expression
[ 1, 2, 3 ] instance of array?

Result:
 true

Example 10.31. Instance of expression
() instance of ()

Result:
 true
