14.2.9. index-of

This function searches a sequence for an atomic value and returns the positions at which it was found.
index-of(item*, anyAtomicType) as integer*
Example 14.53. Filtering redundant values.
index-of(("foo", "bar", "foo", "bar", "foo"), "foo"),
index-of(collection("faq").tags!$$[], "nosql")
