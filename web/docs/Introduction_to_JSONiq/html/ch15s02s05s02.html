15.2.5.2. string-to-codepoints

This function converts a string to the sequence of its Unicode codepoints.
string-to-codepoints(string?) as integer*
Example 15.22. Getting the codepoints of a string.
string-to-codepoints("NoSQL")

Result:
 78
 111
 83
 81
 76
