15.2.5.15. substring-after

This function extract the part of the first parameter that comes after the first occurrence of the second parameter (the empty sequence is considered as "").
substring-after(string?, string?) as string
Example 15.35. Extracting a substring from the first occurrence of another one.
substring-after("NoSQL", "SQL"),
substring-after("NoSQL", "No"),
substring-after("NoSQL", ()),
substring-after("", "SQL")

Result:
 ""
 "SQL"
 "NoSQL"
 ""
