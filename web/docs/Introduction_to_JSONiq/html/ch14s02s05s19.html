14.2.5.19. resolve-uri

Resolves a relative URI against an absolute URI.
resolve-uri(string?, string) as anyURI*
Example 14.39. Resolving a URI.
resolve-uri("types", "http://www.jsoniq.org/"),
resolve-uri(
    "http://www.example.com/types",
    "http://www.jsoniq.org/"
)

Result:
 "http://www.jsoniq.org/types"
 "http://www.example.com/types"
