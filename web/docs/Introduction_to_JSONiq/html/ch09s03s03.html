9.3.3. Castable expressions

Figure 9.14. CastableExpr

A castable expression checks whether a JSONiq value can be cast to a given atomic type and returns true or false accordingly. It can be used before actually casting to that type.
Example 9.51. Castable as expression
"1" castable as integer
Result: 
true

Example 9.52. Castable as expression
"foo" castable as integer
Result: 
false

Example 9.53. Castable as expression
"2013-04-02" castable as date
Result: 
true

Example 9.54. Castable as expression
() castable as date
Result: 
false

Example 9.55. Castable as expression
("2013-04-02", "2013-04-03") castable as date
Result: 
false

The question mark allows for an empty sequence.
Example 9.56. Castable as expression
() castable as date?
Result: 
true
