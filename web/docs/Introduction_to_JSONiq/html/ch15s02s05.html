15.2.5. Functions on Strings

15.2.5.1. codepoints-to-string

This function converts a sequence Unicode codepoints to a string.
codepoints-to-string(integer*) as string
Example 15.21. Building a string from its codepoints
codepoints-to-string((78, 111, 83, 81, 76))

Result:
 "NoSQL"
