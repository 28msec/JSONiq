14.2.4.4. round

This function rounds its input to a certain precision (upwards in case of a tie), to an integer if no precision is provided.
round(numeric?) as numeric?
round(numeric?, integer) as numeric?
Example 14.15. Rounding a number.
round(-2.5145, 3),
round(-2.3)

Result:
 -2.514
 -2
