15.2.5.4. concat

This function concats atomics (as strings) together (like || ).
concat(anyAtomicType?, anyAtomicType?, ...) as string
Example 15.24. Concatenating atomics.
  concat("foo", 1, true, "bar", ())

Result:
 "foo1truebar"
