10.8. Map Operator

JSONiq provides a shortcut for a for-return construct, automatically binding each item in the left-hand-side sequence to the context item.
Example 10.21. A simple map.
(1 to 10) ! ($$ * 2)
Results:

4

Example 10.22. An equivalent query.
for $i in 1 to 10
return $i * 2
