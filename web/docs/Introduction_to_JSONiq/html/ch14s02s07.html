14.2.7. unordered

This function returns the sequence in an order that may be different from its original order. It can be used to give more leeway for optimizations when the order does not matter to the user.
unordered(item*) as item*
Example 14.51. Telling the optimizer that the order of a sequence does not matter.
unordered(("foo", "bar")),
unordered(1 to 10),
unordered(())

Result:
 "foo"
 "bar"
